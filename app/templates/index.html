About

Alfie is a Python script that generates LaTeX files and compiles them with XeTeX. The script was made by Joakim Hertze and turned into a web app by Hagen Trinter. Alfie is available on GitHub. The web app is built with Python, Flask, Bootstrap and Docker. The source code is also available on GitHub.
